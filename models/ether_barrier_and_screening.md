# Модели: Барьер и экранирование в эфирной физике

> "Кулоновский барьер" = порог вязкости/сжатия эфира. Экранирование = решётка как линза.

---

## Стандартная модель vs Эфирная модель

### Кулоновский барьер

**Стандартно:**
```
V_C(r) = k * Z1 * Z2 * e^2 / r
V_C(D-D) ~ 400 кэВ в вакууме
```
Два "заряженных" ядра отталкиваются. Для слияния нужна энергия >> барьера, или квантовое туннелирование.

**Проблема:** при 300K энергия ~0.025 эВ. Вероятность туннелирования: ~10^(-2700). LENR невозможен.

**Эфирная модель:**
```
Барьер = сопротивление эфира при сжатии двух вихрей
B_eff(r) = B_0 * f(rho_ether, v_flow, viscosity)
```

Барьер -- не электростатическое отталкивание зарядов, а:
1. **Вязкость эфира** между двумя вихрями (ядрами)
2. **Давление обратного потока** при сближении
3. Зависит от **локальных условий** среды, а не от фундаментальных констант

## Экранирование как линзирование

### Стандартная модель экранирования

```
V_eff(r) = V_C(r) * exp(-r / lambda_D) - U_s
```
Электроны решётки "экранируют" заряд ядра. Эффективный барьер снижается на U_s.

**Проблема:** экранирование в PdO (600 эВ) в 20 раз больше предсказания адиабатической модели (~30 эВ).

### Эфирная модель: решётка как линза

Кристаллическая решётка -- периодическая структура уплотнений эфира. Она действует как **линза** для эфирных потоков:

```
              без решётки             в решётке

  D ---->  D                D --[====]---> D
  (равномерный поток)       (сфокусированный поток)

  Барьер: высокий           Барьер: низкий
  (вязкость однородна)      (вихрь "проталкивается" потоком)
```

### Почему PdO лучше Pd?

| Параметр | Pd | PdO | Эфирная причина |
|---------|-----|-----|----------------|
| Us (эВ) | 310 | 600 | Оксидный слой -- дополнительный градиент |
| Структура | FCC | FCC + O | Кислород создаёт "каналы" в эфирной решётке |
| Плотность электронов | 0.12 А^(-3) | выше | Более плотная эфирная среда |

Оксидный слой на Pd создаёт **двойную линзу**: Pd-решётка фокусирует, O-слой дополнительно сжимает поток. Это аналогично тому, как многослойная оптическая линза фокусирует свет сильнее однослойной.

## Математическая модель

### Эфирный барьер

```python
def ether_barrier(r, rho_lattice, v_flow, viscosity):
    """
    Барьер в эфирной модели.

    r             -- расстояние между центрами вихрей
    rho_lattice   -- плотность эфира в решётке
    v_flow        -- скорость эфирного потока (определяется решёткой)
    viscosity     -- вязкость эфира (зависит от температуры)
    """
    # Давление обратного потока
    P_back = 0.5 * rho_lattice * v_flow**2

    # Вязкое сопротивление
    F_viscous = viscosity * v_flow / r

    # Фокусировка решёткой (линзирование)
    focusing = rho_lattice / rho_vacuum

    # Эффективный барьер
    B_eff = (P_back + F_viscous) / focusing

    return B_eff
```

### Связь с ML-моделью

В нашей ML-симуляции три физических режима дают три оценки барьера:

| Режим | Формула барьера | Физический смысл |
|-------|----------------|-----------------|
| Maxwell | V_C * exp(-r/lambda) - Us | Электростатика + экранирование |
| Coulomb Original | k * rho1 * rho2 / r^2 | Взаимодействие масс электричества |
| Cherepanov | B_eff(rho, v, eta) | Сопротивление эфира при сжатии |

SHAP-анализ покажет, какой режим лучше соответствует экспериментам.

## Предсказания эфирной модели

1. **PdO > Pd > Ni > Fe > Ti > Au** по экранированию -- совпадает с экспериментом
2. **Многослойные структуры** (Pd/CaO/Pd) дают больший эффект -- подтверждено Iwamura
3. **Температура слабо влияет** (< 10%) -- подтверждено Kasagi
4. **Нанопорошки** дают бОльший эффект из-за большей поверхности линзирования
5. **SUS304 + H2** даёт нейтроны при нагреве -- сталь тоже линза, но слабее Pd
