# Cold Star: Физические основы

> Фундамент Cold Star -- управляемые ядерные процессы в кристаллической решётке при комнатной температуре.

---

## Почему "Cold Star"

Звезда -- это ядерный реактор. Температура ядра Солнца ~15 млн К. Реакции удерживаются гравитацией.

**Cold Star** -- ядерный реактор при 300-800К. Реакции удерживаются структурой кристаллической решётки. "Холодная звезда" -- потому что ядерный синтез без миллионов градусов.

---

## Экспериментальная база

### Доказано (peer-reviewed):

| Факт | Измерение | Источник |
|------|----------|---------|
| Аномальное тепло | 20-240 Вт/см3, COP до 40x | Fleischmann-Pons 1989 |
| Тепло после выключения | 114 МДж из 100г Pd (15 дней) | Mizuno 1991 |
| Трансмутации | Cs->Pr, Sr->Mo, Ba->Sm | Iwamura/Mitsubishi 2002 |
| Изменение изотопов | Cr-52: 84% -> 51% | Mizuno 1998 |
| Нейтроны из стали+H2 | 0.7 МэВ пик, порог 440 C | Mizuno 2025 |
| COP > 3 | 183-209 Вт, COP 3.76-3.91 | Constantan-D2 2025 |
| Тетраэдральная конденсация | 4D -> 2*4He, 47.6 МэВ | Takahashi/TSC теория |

### Что это значит для Cold Star

1. **Энергия есть** -- 114 МДж из 100г (доказано)
2. **COP > 1 достижим** -- 3.76x (доказано)
3. **Трансмутации программируемы** -- Cs->Pr воспроизведён Toyota
4. **Не нужен дейтерий** -- SUS304 + H2 тоже работает
5. **Не нужна плазма** -- всё при 300-800К

## Три уровня физики Cold Star

### Уровень 1: Решётка как линза
```
Pd/CaO многослойная структура
        |
        v  Фокусирует эфирные потоки
        |
        v  Снижает барьер (600 эВ экранирование в PdO)
        |
        v  Разрешает ядерные реакции при kT ~ 0.025 эВ
```

### Уровень 2: Ядерные реакции
```
D + D -> 4He + 23.8 МэВ (доминирующий канал в LENR)
4D -> 8Be* -> 2*4He + 47.6 МэВ (TSC канал)
A(Z) + nD -> A(Z+2n) (трансмутации)
```

### Уровень 3: Тепло + работа
```
Тепловая мощность: 2.1-240 Вт/см3 (измерено)
Плотность энергии: 1.14 ГДж/кг (измерено)
Для сравнения:
  - Бензин: 46 МДж/кг (в 25 раз меньше)
  - Li-ion батарея: 0.9 МДж/кг (в 1200 раз меньше)
  - Уран-235 (деление): 82 ТДж/кг (в 72000 раз больше)
```

Cold Star -- промежуточная ступень: в 1000 раз плотнее химии, без радиации деления.

## Материалы-кандидаты

| Материал | Экранирование (эВ) | Преимущество | Недостаток |
|---------|-------------------|-------------|-----------|
| PdO | 600 | Максимальный эффект | Дорогой |
| Pd/CaO | ~800 (оценка) | Трансмутации доказаны | Сложная фабрикация |
| Pd*Ni/ZrO2 нано | -- | 110 Вт burst (Kitamura) | Нестабильный |
| Ni-mesh + Pd coating | ~420 | Дёшево, COP 1.2-1.5 | Низкий COP |
| SUS304 + H2 | ~200 | Доступный, без дейтерия | Только нейтроны |
| Constantan + D2 | -- | COP 3.76 | Мало данных |
